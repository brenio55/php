SQL Server é um gerenciador de SQL, porém criado para empresas. Ele tem os mesmos recursos que o MySQL, só que é mais rápido quando comparado a mais usuários.

Existem tipos de usuário no SQL, com permissões específicas, etc.

O padrão para o SQL SERVER é: Login: SA. Senha: root.

Para trabalhar com o SQL Server, o PHP precisa de mais algumas extensões para funcionar.

É necessário sempre copiar novas extensões para a pasta ext da pasta php. 

Então, adiciona o extension=nomedoarquivo no php.ini
